# Cloud-Native Observability Lab

A local cloud-native observability lab for a Cloud-Native **Junior Software Developer - Observability** technical review.

This repository will contain:

- a Go HTTP service;
- Prometheus metrics;
- OpenTelemetry traces;
- structured logs;
- Grafana dashboards;
- Loki, Tempo, Alertmanager and optional Mimir notes;
- Docker Compose quickstart;
- local Kubernetes manifests;
- Juju/charms/Day 1/Day 2 documentation.

The final README is completed in the last commit.
